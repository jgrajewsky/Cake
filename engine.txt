Component:
    -Can be attached to Entity
    -Used for storing data

Behaviour (extends Component):
    -Can use Events

System:
    -Cannot be attached to Entity
    -Is static

Component -> Behaviour -> Raycastable -> Collider

Component -> Behaviour -> Raycastable -> ImageRenderer, TextRenderer

Asset Type:
    -Mesh
    -Image -> Texture -> Sprite
    -Font
    -SVG

Graphic:
    -Color
    -Gradient
    -Image
    -Texture
    -Sprite
    -SVG

Renderers:
    -MeshRenderer (Renders Mesh using a Material)
    -GraphicRenderer (Renders given Graphic)
    -TextRenderer (Renders a string of text using a Font)

Events:
    Behaviour:
        -onCreate
        -onAttach
        -onUpdate
        -onDetach
        -onDestroy
        -onReset
    Collider:
        -onCollisionEnter
        -onCollisionExit
        -onCollisionStay
    Raycastable:
        -onPointerEnter
        -onPointerMove
        -onPointerExit
        -onPointerDown
        -onPointerUp
        -onPointerClick
        -onMouseWheel
        -onFocus
        -onBlur
        -onKeyDown
        -onKeyUp
        -onKeyPress
        -onKeyHold
        -onInput
        -onScroll
